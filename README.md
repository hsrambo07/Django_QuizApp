## About the project 

Quiz Application made in django using REST APIs, and integration of swagger docs,etc.

## To run the program

```

 pip3 install -r requirements.txt
 python3 manage.py runserver
 or
 - create super user
 python3 manage.py createsuperuser
 python3 manage.py runserver

```
